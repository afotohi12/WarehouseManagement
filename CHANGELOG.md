- Delphi application foundation

- Database migration engine

- Password hashing service
- User session management
- Initial warehouse management database schema
- Product management database schema
- Role and permission database schema

### Database

- Added version tracking table (`DatabaseVersion`)

- Added automatic database initialization

### Project Structure

- Created modular project architecture
- Added Core module
- Added Data module

- Added Security module
- Added Forms module
